Wikipedia Nissan Página oficial Review en YouTube

El Nissan Patrol es un automóvil todoterreno con tracción en las cuatro ruedas, producido por el fabricante japonés Nissan en Japón desde el año 1951 hasta la fecha. El Patrol se vende especialmente en los mercados de Australia, Nueva Zelanda, América Central, Sudamérica, Sudáfrica, Europa y el Oriente Medio. Su rival directo es el Toyota Land Cruiser..

En la página web sobre el Nissan Patrol, se aplicaron las siguientes técnicas:

Estructura HTML:
Se utilizó HTML5 para estructurar el contenido de la página. Se dividió en secciones como encabezado, barra de navegación, contenido principal, contenido lateral y pie de página. Se añadieron imágenes en la sección del encabezado, utilizando la etiqueta <img>. Cada imagen se le asignó una clase específica para poder aplicar estilos de alineación en CSS.

Estilización con CSS3:
Se empleó CSS para dar estilo a la página. Se creó un archivo de estilos (Style.css) que se enlazó al documento HTML. Para alinear las imágenes en el encabezado a la izquierda y derecha, se usaron clases como .logo-left y .logo-right con propiedades de estilo como float: left y float: right, respectivamente. Se diseñó el contenido lateral (aside) con un fondo de color gris claro y se utilizó la propiedad display: flex con align-items: center y justify-content: center para centrar dinámicamente el GIF, permitiendo que se redimensione automáticamente con el cambio del tamaño de la ventana del navegador.

Enfoque Responsivo:
Se aplicaron propiedades CSS como max-width: 100% y max-height: 100% en la clase .gift-aside para asegurar que el GIF se ajustara dentro del aside sin desbordarse, permitiendo que mantenga sus proporciones y se redimensione según el espacio disponible.

Compatibilidad con Diferentes Tamaños de Pantalla:
El diseño se adaptó para responder a diferentes tamaños de pantalla. Por ejemplo, las imágenes del encabezado se configuraron para ajustarse y mantener su posición relativa al tamaño del contenedor del encabezado. Este enfoque combina HTML5 para la estructura del contenido con CSS3 para el diseño y la presentación, permitiendo que la página se adapte y sea más flexible al redimensionar la ventana del navegador, lo que asegura una mejor experiencia del usuario en diferentes dispositivos y tamaños de pantalla.
